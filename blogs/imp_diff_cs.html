Difference Between sort() and sorted()

Function What it does Example Output Example
sort() Sorts the list in place (modifies the original list) and returns None. my_list = [3, 1, 4]; my_list.sort() [1, 3, 4]
sorted() Returns a new sorted list without modifying the original list. new_list = sorted([3, 1, 4]) [1, 3, 4]

Difference Between lstrip(), rstrip(), and strip()

Function What it does Example Output Example
lstrip() Removes leading (left-side) whitespace from a string. " hello ".lstrip() "hello "
rstrip() Removes trailing (right-side) whitespace from a string. " hello ".rstrip() " hello"
strip() Removes both leading and trailing whitespace from a string. " hello ".strip() "hello"

Network Type Full Form Coverage Area Example

Difference Between seek() and tell()

Function What it does Example Output Example
seek() Moves the file cursor to a specific position. file.seek(5) Cursor moves to position 5 in the file.
tell() Returns the current position of the file cursor. pos = file.tell() Returns current cursor position, e.g., 5.

Difference Between read(), readline(), and readlines()

Function What it does Example Output Example
read() Reads the entire file or a specific number of characters. file.read() Returns whole file content as a string.
readline() Reads a single line from the file. file.readline() Returns one line as a string.
readlines() Reads all lines and returns a list of strings. file.readlines() Returns list: ["Line 1", "Line 2"]

Difference Between write() and writelines()

Function What it does Example Output Example
write() Writes a single string to a file. file.write("Hello World") Writes "Hello World" to the file.
writelines() Writes a list of strings to a file. file.writelines(["Line 1\n", "Line 2\n"]) Writes multiple lines to the file.

Difference Between dump() and load()

Function What it does Example Output Example
dump() Serializes Python objects and writes them to a file. json.dump(data, file) Stores JSON data in a file.
load() Reads JSON data from a file and deserializes it into a Python object. data = json.load(file) Loads JSON data into a Python dictionary.

Term Full Form Measurement Common Use

Mbps Megabits per second Measures data transfer speed in megabits Used for internet speed and bandwidth

Difference Between fetchone() and fetchall()

Function What it does Example Output Example
fetchone() Retrieves the next row from the result set. cursor.fetchone() Returns a single row as a tuple.
fetchall() Retrieves all remaining rows from the result set. cursor.fetchall() Returns a list of tuples containing all rows.

Difference Between execute() and executemany()

Function What it does Example Output Example
execute() Executes a single SQL statement. cursor.execute("INSERT INTO table VALUES (?, ?)", (1, "John")) Inserts one row into the table.
executemany() Executes the same SQL statement multiple times with different parameters. cursor.executemany("INSERT INTO table VALUES (?, ?)", [(1, "John"), (2, "Jane")]) Inserts multiple rows into the table.
