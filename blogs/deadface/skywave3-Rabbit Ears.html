SKYWAVE 3: RABBIT EARS

SQL • Aggregation/Joins • 80 pts

Overview

“SkyWave 3: Rabbit Ears” by syyntax asks for the most frequent antenna type used on towers managed by Florian Olyff. This requires joins and a grouped count.

Challenge Details
Points80
Creatorsyyntax
ObjectiveFind most common antenna type for Olyff
Submitflag{antenna_name number}

Continue from previous SkyWave challenges.

SQL — most common antenna type for Olyff
SELECT a.antenna_name, COUNT(*) AS antenna_count
FROM Towers t
JOIN Tower_Sectors ts ON t.tower_id = ts.tower_id
JOIN Antennas a ON ts.antenna_id = a.antenna_id
WHERE t.operator_id = 4  -- Florian Olyff's operator_id
GROUP BY a.antenna_name
ORDER BY antenna_count DESC
LIMIT 1;
Result

The most common antenna is Multiple Input Multiple Output (MIMO) with a count of 3.
