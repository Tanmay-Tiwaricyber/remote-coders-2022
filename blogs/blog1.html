How to Hide Folder or Files from Windows?

Step 1: Download This File

Step 2: Run this .bat file

Step 3: A folder is created named 'private'

Step 4: Edit the .bat file, enter your password where it says 'Enter Your Password'

Step 5: Place your files and folders in the 'private' folder

Step 6: Close the 'private' folder

Step 7: Run the .bat file and press Y for yes

Step 8: To unlock and unhide files, run the .bat file again and enter your password
